# セットアップ方法

## リポジトリをクローンする
```
git clone https://github.com/hirotomoki12345/youtube.git
```

## ディレクトリに移動
```
cd youtube
```
## 必要なパッケージをインストール
```
npm i
```
## youtube-dl-exec パッケージをインストール
```
npm install youtube-dl-exec --save
```
## アプリケーションを起動
```
npm start
```
